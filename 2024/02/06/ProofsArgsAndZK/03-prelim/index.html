交互式证明

对于映射 f:{0,1}n→Rf:\{0, 1\}^n\to \mathcal{R}f:{0,1}n→R，一个 kkk-消息的交互式证明系统（IP）包含：

  • 一个随机化的 Verifier 算法 V\mathcal{V}V
  • 一个确定性的 Prover 算法 P\mathcal{P}P

交互流程：

  • 公共输入 x∈{0,1}nx\in \{0, 1\}^nx∈{0,1}n
  • P\mathcal{P}P 声明 f(x)=yf(x)=yf(x)=y
  • P\mathcal{P}P 与 V\mathcal{V}V 交替发送信息 m1,m2,…,mkm_1, m_2,\dots, m_km1​,m2​,…,mk​
  • V\mathcal{V}V 收到最后一条信息后输出 0/10/10/1

P\mathcal{P}P 与 V\mathcal{V}V 可被视作关于 (x,m1,…,mi)(x, m_1, \dots, m_i)(x,m1​,…,mi​) 到 mi+1m_{i+1}mi+1​ 的函数，V\mathcal{V}V 还可以使用内部的随机性。

交互记录（transcript）：(y,m1,…,mk)(y, m_1, \dots, m_k)(y,m1​,…,mk​)

确定掷币结果 rrr 后可以定义输出 out(V,x,r,P)=0/1\mathrm{out}(\mathcal{V}, x, r, \mathcal{P})=0/1out(V,x,r,P)=0/1，其为 xxx 的确定性函数。

完整性误差 δc\delta_cδc​：对于任意 xxx，有

Pr⁡r[out(V,x,r,P)]≥1−δc\Pr\limits_{r}[\mathrm{out}(\mathcal{V}, x, r, \mathcal{P})]\ge 1-\delta_c rPr​[out(V,x,r,P)]≥1−δc​

可靠性误差 δs\delta_sδs​：对于任意 xxx，任意确定性算法 P′\mathcal{P}'P′，若 P′\mathcal{P}'P′ 发送的 y≠f(x)y\neq f(x)y=f(x)，有

Pr⁡r[out(V,x,r,P′)]≤δs\Pr\limits_{r}[\mathrm{out}(\mathcal{V}, x, r, \mathcal{P}')]\le \delta_s rPr​[out(V,x,r,P′)]≤δs​

只要两侧错误率都 ≤1/3\le 1/3≤1/3，则该系统是有效的。

⌈k/2⌉\lceil k/2\rceil⌈k/2⌉ 称为轮复杂度（round-complexity）。

论证系统

如果只对多项式时间的所有 P′\mathcal{P}'P′ 要求可靠性，这样的可靠性称为计算可靠性，而上文中的可靠性称为信息论可靠性。

在论证系统中，许多密码学算法可以得到应用。

讨论

  • 任何 δc≤1/3\delta_c\le 1/3δc​≤1/3 的 IP 均可转化为完美完整性的 IP，且对 Verifier 端开销的影响是多项式级别的。
  • 任何 δs≤1/3\delta_s\le 1/3δs​≤1/3 的 IP 仅需简单重复即可提高可靠性。
  • 对于任何一个掷币结果保密的 IP，其均可由一个掷币结果公开的 IP 模拟，开销影响是多项式级别的。
  • 随机性的 P\mathcal{P}P 与确定性的 P\mathcal{P}P 能力相当。因为如果其掷币结果使得其说服 V\mathcal{V}V 的概率达到 ppp，必存在一个掷币结果序列，固定该序列得到的确定性策略说服 V\mathcal{V}V 的概率至少为 ppp。

对于一个语言 L⊂{0,1}n\mathcal{L}\subset \{0, 1\}^nL⊂{0,1}n，亦可定义 IP ：

存在 fff，使得：

  • 任取 x∈Lx\in \mathcal{L}x∈L，存在 P\mathcal{P}P 使得 V\mathcal{V}V 以高概率接受 xxx；
  • 任取 x∉Lx\not\in \mathcal{L}x∈L，对于任意 P\mathcal{P}P，V\mathcal{V}V 以高概率拒绝 xxx。

注意此时交互式证明系统的定义不要求证明 x∉Lx\not \in \mathcal{L}x∈L。

定义复杂性类 IP\mathsf{IP}IP 为所有能由交互式证明系统在验证方 V\mathcal{V}V 使用多项式时间内证明的语言 L\mathcal{L}L 的集合。

IP=PSPACE\mathsf{IP}=\mathsf{PSPACE}IP=PSPACE。

低阶多线性拓展

对于多元函数 f:{0,1}v→Ff:\{0, 1\}^v\to \mathbb{F}f:{0,1}v→F 考虑，只需取 v=log⁡nv=\log nv=logn 即可容纳 nnn 个点值。

如果 vvv 元多项式 g:Fv→Fg:\mathbb{F}^v\to \mathbb{F}g:Fv→F 与 fff 在 fff 的定义域上的点值吻合，可视 ggg 为 fff 的拓展。当 v≪∣F∣v\ll\left|\mathbb{F}\right|v≪∣F∣ 时，这样的拓展是 distance-amplifying 的。

如果 ggg 中每一项中每个未定元的次数均不超过 1，则称 ggg 是 fff 的多线性拓展。

任何一个函数 f:{0,1}v→Ff:\{0, 1\}^v\to \mathbb{F}f:{0,1}v→F 在 F\mathbb{F}F 上存在唯一的多线性拓展（MLE），记为 f~\tilde ff~​。

记

χw(x1,…,xv)≔∏i=1v(xiwi+(1−xi)(1−wi))\chi_w(x_1, \dots, x_v)\coloneqq \prod\limits_{i=1}^v(x_iw_i+(1-x_i)(1-w_i))χw​(x1​,…,xv​):=i=1∏v​(xi​wi​+(1−xi​)(1−wi​))

则

f~(x1,…,xv)=∑w∈{0,1}vf(w)⋅χw(x1,…,xv)\tilde f(x_1, \dots, x_v)=\sum\limits_{w\in \{0, 1\}^v}f(w)\cdot \chi_w(x_1, \dots, x_v)f~​(x1​,…,xv​)=w∈{0,1}v∑​f(w)⋅χw​(x1​,…,xv​)

{χw∣w∈{0,1}v}\{\chi_w\mid w\in \{0, 1\}^v\}{χw​∣w∈{0,1}v} 称为插值集合 {0,1}v\{0, 1\}^v{0,1}v 上的多元 Lagrange 多项式基。

记 n=2vn=2^vn=2v。对于给定 fff 及 rrr，直接计算 f~(r)\tilde f(r)f~​(r) 显然需要 O(nlog⁡n)\mathcal{O}(n\log n)O(nlogn) 时间及 O(log⁡n)\mathcal{O}(\log n)O(logn) 空间。
